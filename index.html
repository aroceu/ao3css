ao3-style.css

What is this?

A pre-written stylesheet for fics from AO3 directly downloaded as .html files.

Who is this for?

Fanfiction writers who would like to upload their fanfiction that they've already uploaded to AO3 to an independent website, such as ones hosted by Neocities, Nekoweb, or Teacake.

Though AO3 conveniently has your fic converted from webpage to a viewable .html file, it's not exactly the prettiest. Or most readable, depending on who you are. (See this page as an unstyled example.)

For writers who are concerned with such things, this stylesheet is an easy and quick solution to that while also making the backup of your fanfiction on your own site styled to your personal tastes.

How can I use it?
  1. Select all + copy ao3-style.css and paste it into your own ao3-style.css file. Alternatively, you can download the page as-is.
  2. Download your fics from AO3 in HTML format. Put them in the same folder as ao3-style.css
    • This can be done by going to: AO3 Fic Page > Download > HTML, or
    • Using the ao3downloader python script made by nianeyna.
      • Do my fics have to be in the same folder as ao3-style.css?
      It would be easier for the purposes of this instruction, but it'll still work if you spread your fic across different folders, or put this stylesheet elsewhere. I won't go into it because it depends on your specific and desired structure. To learn more, read about relative file paths.
  3. Open each fic .html file in your text editor. (You don't have to do them all at once, but you will need to repeat this step for all of them eventually.)
    1. Copy this code:
      2. <link rel="stylesheet" href="ao3-style.css"> <!-- this adds the stylesheet -->
<meta name="viewport" content="width=device-width, initial-scale=1"> <!-- this allows the stylesheet for mobile/tablets to work -->
    2. At the top of the .html file from line 7 to line 22, there should be this code:
        <style type="text/css">
    p.message { text-align: center; }
    .meta h1 { font-size: 1.5em; text-align: center; }
    .meta h2 { font-size: 1.25em; text-align: center; }
    .meta h2 { page-break-before: always; }
    .meta .byline { text-align: center; }
    .meta dl.tags { border: 1px solid; padding: 1em; }
    .meta dd { margin: -1em 0 0 10em; }
    .meta .endnote-link { font-size: .8em; }
    /* List child related works under the labeling dt */
    #afterword .meta dd { margin: 1em 0 0 1em; }
    #chapters { font-family: "Nimbus Roman No9 L", "Times New Roman", serif; padding: 1em; }
    .userstuff { font-family: "Nimbus Roman No9 L", "Times New Roman", serif; padding: 1em; }
    /* Invisible headings to help Calibre make a Table of Contents */
    .toc-heading { display: none; }
  </style>
    3. Select it all, and replace it by pasting the code you copied earlier. Do this for all of your fic.
  4. Now you can style! Open ao3-style.css in your text editor.
  5. Change everything in the :root {} brackets to your heart's content.
    • You will need to be familiar with hex color codes and using fonts in CSS, but everything else should be self-explanatory.
    • You can test your changes by opening any of your fic's .html files in your web browser.
    • If you aren't seeing the changes right away, hard refresh (shift+ctrl+R on most browsers) to clear your cache.
    • You can also check your code against the samples (see above) to make sure everything looks good.
  6. Upload the folder containing your fic .html files as well as your ao3-style.css to your website. Then you're done!
Keep in mind!
  • Do not add additional HTML lines in the <div id="preface"> ... <div class="meta">... ...</div> </div> section. Formatting for the metadata labels (Summary, Notes) is dependent on the order of elements in this div, so adding new elements (or removing them) will mess this up.
  • If you would like to use the same styling as your masterlist or another webpage on your site, call both stylesheets on your fic page, instead of just ao3-style.css. You can do something like this: 
    <link rel="stylesheet" href="style.css"> <!-- this adds the masterlist stylesheet -->
<link rel="stylesheet" href="ao3-style.css"> <!-- this adds the ao3 fic stylesheet -->
<meta name="viewport" content="width=device-width, initial-scale=1"> <!-- this allows the stylesheet for mobile/tablets to work -->
    The last stylesheet that gets called always takes priority, which is why this code orders it second. The first stylesheet can be thought of as the default, while subsequent stylesheets can be thought of as specific customizations.
  • Though I cannot offer 24/7 support for everyone, if you have any concerns about this code specifically, please feel free to contact me and I'll see what I can do to help :)

All customization can be done by modifying only the variables under :root{} in ao3-style.css.
